Serveur domotique Serveur web

Creation serveur domotique sur mon raspberry

Materiel utiliser

  • Un raspberry pi 4 model B 2Gb RAM
  • Carte micro SD samsumg 32Gb
  • Alimentation 2,4A
  • Cable USB B vers USB C
  • Pont Philips Hue
  • Anpoule Philips Hue
  • Interupteur Philips Hue
  • Yeelight Aurora Lightstrip Plus
  • Router WIFI
  • Carble RJ45

Contexte

J'utisse des apareils de marque differente et j'aimerais que mon interupteur puise commander mes lampes de marque differente

Mise en place

Pour cette realisation tout les appereil son neuf et non encore jamais servie

premierement nous allons flashé la carte SD avec le logiceil Balena Etcher en utilisant un ISO de Rasbian Buster Lite,

une fois la carte flashé je crée un ficher sans extention se nomant simplement ssh se fichier permetra au raspberry de se preparer a une copnnection en SSH des son alumage

Maintenemt nous alons configurer le Wifi sur le raspberry pour pouvoir le pose dans un coin dans ma maison, pour cela nous le branchon au routeur avec le cable RJ45, nous pouvoir l'allimenter

Sur notre ordinateur avec le logiciel Mobaxterm dans tools network scanner je lance un scan pour trouver son IP sur le teste vous devriez voir un appareil utilisant le protocol ssh je vais donc lancé une sesion SSH sur l'IP du rasberry (Mobaxterm me permet de retenir l'idantifiant de connection et le mot de passe)

par defaut l'idantifiant est :

Identifiant : pi
mot de passe : raspberry

je vous recommande fortement de changé le mot de passe et de le remplacer par un autre plus securiser si un message vous indice vous idique un probleme avec le WIFI suivé les instructeur
